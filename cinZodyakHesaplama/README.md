
Çin zodyakları hesaplanırken kişinin doğum yuılının 12 ile bölümünden kalana bakılır. Yani Doğum tarihi % 12 == ? sonucu.

Kalanlara göre aşağıdaki gibi değerlendirilir.

* 0  => Horoz
* 1  => Maymun
* 2  => Köpek
* 3  => Domuz
* 4  => Fare
* 5  => Öküz
* 6  => Kaplan
* 7  => Tavşan
* 8  => Ejderha
* 9  => Yılan
* 10 => At
* 11 => Koyun
* 12 => Keçi
